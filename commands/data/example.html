"About" tag can be used for description the recipe.
Text from "description" tag will be show in Menu dialog "Selected" tab.
If you selected this recipe on Menu column "Recipes" tab in left panel.

The description tag supports markdown code, and you can use hyperlinks for
web and recipe call. For example:

  • Anteater Sorceforge Project
  • 'About' Recipe

You can also add information about the recipe creator and attach the necessary links.

Note: This tag is processed only Desktop UI application version.
